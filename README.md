# Missão Python

Este repositório reúne exercícios desenvolvidos durante uma **missão prática de Python**, com foco em lógica de programação e fundamentos da linguagem.

O objetivo é simples: resolver problemas com código claro, funcional e bem pensado. Menos truque, mais base.

---

## Sobre

Os exercícios seguem uma progressão de dificuldade e trabalham habilidades essenciais para quem está aprendendo a programar em Python, como:

- raciocínio lógico  
- autonomia na resolução de problemas  
- leitura e interpretação de enunciados  
- escrita de código simples e organizado  

Cada script representa um desafio resolvido e faz parte do meu processo de aprendizado e evolução na linguagem.

Este repositório serve como:
- registro de estudo  
- material de revisão  
- demonstração prática de aprendizado  

Código direto. Sem enfeite.

---

## Conteúdos Trabalhados

- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`while`, `for`)
- Entrada e saída de dados
- Operações matemáticas e lógicas
- Organização e leitura de código
- Pensamento algorítmico

Base forte antes de abstração.
